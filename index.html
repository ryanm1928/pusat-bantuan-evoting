User Cara voting

 1] Silakan akses link yang diberikan panitia.
2] Kemudian pilih kelas sesuai dengan kelas kamu.
3] Kemudian pilih nama kamu.
4] Kemudian masukan password yang telah di berikan admin.
5] Lalu klik Login.

4] Kemudian klik lakukan voting.
5] Akan muncul kandidat yang akan maju menjadi pemimpin kemudian, tekan vote pada calon pilihan kalian.
6] Setelah menekan vote akan mundul pop up verifikasi lalu klik pilih.
7] Hore... voting berhasil dengan tanda ceklist hijau berhasil
8] Kalian bisa logout dengah menekan Bars (garis 3) lalu klik logout.

Admin Membuat dan Mengedit Voting

Evoting memberikan fitur membuat voting untuk user melakukan voting

Membuat voting:
  • Untuk membuat voting anda bisa menekan menu Buat Voting
  • Lalu aplikasi akan meminta jumlah kadidat yang akan maju Min:2 | Max:6 Kadidat
  • Setelah memasukan jumlah kadidat anda dapat mengisi form yang telah di sediakan
  • Setelah selesai mengisi form anda dapat menekan tombol

Evoting memberikan fitur edit voting bila ada kesalahan dalam pengisian form

Edit voting:
  • Untuk mengedit voting anda bisa menekan menu Home
  • Anda akan melihat table data voting yang telah anda buat
  • Pada table anda dapat menekan icon
  • Lalu anda akan di sediakan form untuk mengedit data
Admin Melihat Hasil Voting

Evoting memberikan fitur melihat hasil voting telah di lakukan user

  • Untuk melihat voting anda bisa menekan menu Lihat Voting
  • Anda akan melihat table data voting yang telah anda buat
  • Pada table Action anda dapat menekan icon untuk melihat hasil voting
Admin Manajemen User

Evoting memberikan fitur manajemen user yang berguna untuk mengatur data user

Manajemen User:
  • Untuk memanajemen user anda bisa menekan menu Statistik Pengguna
  • Pada ada form pencarian anda bisa gunakan untuk mencari data user
  • Di bawah form pencarian ada 3 card untuk memonitoring user
  • Kemudian pada table berisi data user yang berguna untuk menampilkan data user
  • Pada table status berguna untuk melihat apakah user sudah melakukan voting / belum
  • Pada table Action anda dapat menekan icon Untuk mengedit data user
  • Pada table Action anda dapat menekan icon Untuk menghapus data user
Membuat dan import akun User

Evoting memberikan fitur membuat akun user yang di gunakan untuk melakukan voting

Membuat akun user:
  • Untuk membuat akun user anda bisa menekan menu Tambah Pengguna
  • Kemudian akan di sediakan form yang bisa anda bisa gunakan untuk membuat akun user

Evoting memberikan fitur import user apabila ingin membuat akun yang banyak sekaligus dengan menggunakan MS excel

import user:
  • Anda dapat menekan tombol
  • Kemudian anda akan di tampilkan syarat dan ketentuan import data
  • Setelah memahami syarat dan ketentuan import data anda dapat melakukan import data
About

Aplikasi E-Voting berbasis website yang di bangun menggunakan Laravel versi 8 untuk backend & Boostrap versi 4 untuk Frontend, Aplikasi ini bisa di gunakan untuk Kegiatan voting secara online seperti Pemilihan Ketua OSIS dll. Adapun fitur yang yang ada dalam Aplikasi ini:

Fitur untuk User:
  • Login System User
  • User dapat melakukan voting
  • Validasi User bila melakukan voting
  • User dapat mengirim pesan untuk admin
  • Ubah password User
  • Reponsif
Fitur untuk Admin:
  • Manage data voting
  • Manage data user
  • Admin dapat membalas pesan dari user
  • Validasi untuk admin bila melakukan voting
  • Responsif
  • pdf reporting
